# 第3回 強化学習の基礎

## 概要

強化学習（Reinforcement Learning）の基本的な枠組みについて学習した。

強化学習では、エージェントが環境との相互作用を通じて報酬を獲得し、その総和を最大化する行動を学習する。

---

## 学んだこと

### 強化学習の枠組み

強化学習は以下の要素によって構成される。

- Agent（エージェント）
- Environment（環境）
- Action（行動）
- Reward（報酬）

エージェントは環境から得られる情報をもとに行動を選択し、その結果として報酬を受け取る。

### マルコフ決定過程（MDP）

強化学習の標準的な枠組みとしてマルコフ決定過程（MDP）が用いられる。

MDPは以下の要素で構成される。

- 状態（State）
- 行動（Action）
- 状態遷移確率（Transition）
- 報酬（Reward）

現在の状態と行動によって次の状態が確率的に決定される。

### 方策（Policy）

方策（Policy）とは、各状態においてどの行動を選択するかを定めるルールである。

強化学習の目的は、将来得られる報酬の期待値を最大化する方策を見つけることである。

### Q学習（Q-Learning）

Q学習は代表的な価値ベースの強化学習アルゴリズムである。

各状態と行動の組み合わせに対してQ値（行動価値）を学習し、Q値が最も高い行動を選択することで最適な行動を決定する。

## 演習

Google Colab上でQ学習による迷路探索を実装した。

### 実施内容

- 複数経路のある迷路で最短経路を探索
- 学習パラメータを変更して結果を比較
- Q値の更新過程を確認

---

## 自分の理解

強化学習では正解データを与えるのではなく、報酬を通じて行動を学習する。

サッカーに例えると、選手が「得点につながるプレー」を報酬として試行錯誤を繰り返し、最適な判断を学習していくイメージである。

また、Q学習では現在の報酬だけでなく将来的な報酬も考慮するため、短期的な利益だけでなく長期的な利益を見据えた行動を学習できることが分かった。
